# Symora

**Open-source Detection Engineering Workbench for Sigma Rules**

Symora is an open-source Detection Engineering Workbench designed to help security teams validate, review, score and manage Sigma detection rules before deployment.

---

## Why Symora?

As Sigma rule libraries grow, maintaining quality and consistency becomes increasingly difficult. Manual reviews are time-consuming and often inconsistent.

Symora provides a structured workflow that enables detection engineers to:

- Validate Sigma rules
- Map detections to MITRE ATT&CK
- Assess rule quality
- Calculate risk scores
- Generate professional reports
- Manage a centralized rule repository

The goal is to improve the quality, consistency and maintainability of detection content before it reaches production environments.
